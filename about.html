BritishWeather About

A minimalistic British weather app

Hey there! My name is IronVeil and this is a basic weather site that I made. I wanted to try out JavaScript and using APIs (such as the OpenWeatherMap API found here), so I decided to make a weather app based around Britain, where I live. I also made it look nice, which I'm proud of.


I began this site during the summer holidays, when I went up to see family in Scotland. On the trips there, I wasted my data on building it and I'm proud of what I've made. When sorting out my GitHub at the start of this year, I remembered this project and remade parts of it, and its now better than ever.
